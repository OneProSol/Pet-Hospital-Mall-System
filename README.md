### 一、相关技术

- 前端：`thymeleaf`、`HTML` / `CSS` / `Javascript`。
- 后端：`Springboot`等。
- 数据库：`MySQL`

### 二、开发软件（列出的软件其一均可运行）

- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`

### 三、部署所需环境

- 后端：`jdk1.8`。
- 前端：`node`、`vue-cli` 、`npm`  。
- 数据库：`mysql5.7以上`

### 四、功能描述及功能图展示

系统分为：用户。

用户功能：1. 登录、2. 注册、3. 首页、4. 宠物商城、5.商品详情、6. 医院前沿、7. 养宠小知识、8. 个人中心、9.购物车、10.个人中心
